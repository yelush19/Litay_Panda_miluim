# 🚀 מערכת ניהול תשלומי מילואים - ליטאי

## 📋 מה נדרש:
- Node.js גרסה 16 ומעלה (הורדה מ: https://nodejs.org)

## ⚙️ התקנה - 3 שלבים פשוטים:

### 1️⃣ הורדת Node.js
אם אין לך Node.js:
1. כנס ל: https://nodejs.org
2. הורד את הגרסה המומלצת (LTS)
3. הפעל את ההתקנה
4. אישור שהתקנה תקינה: פתח CMD והקלד `node -v`

### 2️⃣ התקנת המערכת
1. שמור את כל הקבצים בתיקייה `miluim-system`
2. פתח CMD/Terminal בתיקייה
3. הקלד: `npm install`
4. המתן להתקנת החבילות (דקה-שתיים)

### 3️⃣ הפעלת המערכת
1. הקלד: `npm start`
2. המתן להודעה: "✅ השרת פועל על..."
3. פתח דפדפן וכנס ל: http://localhost:3000

## 📱 איך להשתמש:

### ייבוא נתוני מילואים:
1. **דשבורד** → גרור קובץ Excel מקאנו
2. המערכת תזהה עובדים אוטומטית
3. תראי סיכום: כמה עובדים, כמה ימים

### ניהול עובדים:
1. **ניהול עובדים** → רשימת כל העובדים
2. עדכן תעריף יומי לכל עובד
3. הוסף עובד חדש ידנית (אם צריך)

### דוח חודשי:
1. **דוח חודשי** → בחר חודש
2. רואה מי עשה מילואים, מתי, כמה
3. לחץ "יצוא לPDF" לשליחה לביטוח לאומי

### תשלומים והפרשים:
1. **תשלומים והפרשים** → לחץ "הוסף תשלום"
2. בחר עובד + חודש
3. הזן כמה כסף התקבל מביטוח לאומי
4. המערכת תחשב הפרש אוטומטית!

## 🎨 צבעי המערכת:
- ✅ ירוק = פעולה ראשית / הצלחה
- 🔵 כחול בהיר = מידע
- 🟡 כתום = אזהרה / ממתין
- 🔴 אדום = שגיאה / חשוב

## ❓ בעיות נפוצות:

**השרת לא עובד?**
- ודא ש-Node.js מותקן: `node -v`
- רוץ שוב: `npm install` ואז `npm start`

**הדפדפן לא נפתח?**
- פתח ידנית: http://localhost:3000
- נסה דפדפן אחר (Chrome מומלץ)

**הקובץ לא מתייבא?**
- ודא שזה קובץ Excel (.xlsx או .xls)
- בדוק שיש עמודות: תאריך, שם עובד, מחלקה, מילואים

## 💡 טיפים:

1. **גיבוי אוטומטי** - כל הנתונים נשמרים בקובץ `miluim.db`
2. **עבודה רציפה** - סגור והפעל מחדש - הכל נשמר
3. **חיפוש מהיר** - השתמש ב-Ctrl+F בדפדפן
