# 🎖️ מערכת ניהול תשלומי מילואים - ליטאי

## 📋 סקירה כללית

מערכת מתקדמת לניהול תשלומי מילואים עבור משרד רואי החשבון של ליטאי ניהול שירותים.

### ✨ תכונות עיקריות:
- ✅ ייבוא נתונים מקובץ Excel (קאנו)
- ✅ ניהול עובדים ומעקב ימי שירות
- ✅ דוחות חודשיים מפורטים
- ✅ סטטיסטיקות בזמן אמת
- ✅ עיצוב בצבעי המותג של ליטאי

---

## 🚀 התקנה והפעלה

### דרישות מקדימות:
- Node.js גרסה 16 ומעלה

### שלבי התקנה:

#### Windows:
```powershell
# 1. פתח PowerShell בתיקיית המערכת
cd C:\path\to\miluim-system-fixed

# 2. התקן תלויות
npm install

# 3. הפעל את המערכת
npm start
```

#### Mac/Linux:
```bash
# 1. פתח Terminal בתיקיית המערכת
cd /path/to/miluim-system-fixed

# 2. התקן תלויות
npm install

# 3. הפעל את המערכת
npm start
```

### פתיחת המערכת:
לאחר הפעלת השרת, פתח דפדפן וגלוש ל:
```
http://localhost:3000
```

## 📥 ייבוא נתונים

### פורמט קובץ Excel (קאנו):

הקובץ חייב להכיל את העמודות הבאות:

| עמודה | תיאור | דוגמה |
|-------|-------|--------|
| ת.ז. | תעודת זהות של העובד | 123456789 |
| שם פרטי | שם פרטי | יוסי |
| שם משפחה | שם משפחה | כהן |
| תאריך | תאריך שירות | 01.06.2025 |
| ימים | מספר ימי שירות | 5 |
| תעריף יומי | (אופציונלי) | 500 |
| מחלקה | (אופציונלי) | מל"מ |

### פורמטים נתמכים לתאריכים:
- `dd.mm.yyyy` (עברי) → `01.06.2025`
- `mm/dd/yyyy` (אמריקאי) → `06/01/2025`
- מספר סידורי Excel
